# Estadística de Emails (mbox)

Script en Python que analiza archivos tipo **mbox** y genera estadísticas a partir de los correos electrónicos.

## Funcionalidades
- Extracción de emails desde líneas `From`
- Conteo de correos por remitente
- Identificación del remitente con más envíos
- Estadísticas de correos enviados por hora
- Uso de expresiones regulares (regex)

## Requisitos
- Python 3.x

## Ejecución
```bash
python estadistica_emails.py
